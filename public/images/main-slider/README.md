# Images du Slider Principal

## Images à télécharger depuis le site live

Téléchargez les images suivantes depuis https://novaservicess.netlify.app/:

1. **right.png** - Image hero du côté droit (artisan/technicien)
   - Url: https://novaservicess.netlify.app/images/main-slider/right.png
   - Dimension: ~500x600px
   - Placement: Section héro, côté droit du texte

2. **prof.png** - Image de professionnel pour les témoignages
   - Url: https://novaservicess.netlify.app/images/main-slider/prof.png
   - Dimension: ~120x120px
   - Placement: Témoignages (optionnel)

## Instructions

1. Ouvrez https://novaservicess.netlify.app/ dans votre navigateur
2. Faites clic droit sur l'image du héros (artisan) → "Enregistrer l'image sous"
3. Enregistrez-la en tant que `right.png` dans ce dossier
4. Répétez pour les autres images

Les fichiers seront automatiquement référencés dans `/app/page.tsx` une fois placés ici.
